# LifeMarker  
ㅤ  
Vue.js 사용에 익숙해지기 위해 만든 `팀 소개` 컨셉의 홈페이지 입니다.  
간단한 자동응답 기능을 가진 `챗봇`도 직접 구현했습니다.
